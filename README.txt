SETUP GUIDE

1. Install XAMPP
2. Move project folder to C:\xampp\htdocs
3. Start Apache and MySQL from XAMPP
4. Open phpMyAdmin and import database.sql
5. Run project using:
   http://localhost/cyber_incident_reporting_system

PROJECT MODULES
- Employee Incident Reporting
- Threat Alert Monitoring
- Incident Severity Classification
- Investigation Status Tracking
- Secure Database Storage
- AJAX Request-Response Communication

TECHNOLOGIES
PHP, MySQL, AJAX, JavaScript, HTML, CSS
